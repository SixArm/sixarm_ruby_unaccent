Replace a string’s accent characters with ASCII characters.

Based on Perl Text::Unaccent from CPAN.

French Example

  str = "déjà vu";
  str.unaccent => "deja vu";

Greek Example

  str = "νέα" (aka "\x{03AD}\x{03BD}\x{03B1}")
  str.unaccent => "νεα" (aka "\x{03B5}\x{03BD}\x{03B1}")
